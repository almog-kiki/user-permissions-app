
# Users permisions App 

angular CLI 10 , node.js, python, mongodb

## versions:
```
    npm     v6.13.7
    node    v13.10.1
    angular cli v10
    python  v2.7(.17_1)
```

## How to run :

 ### NOTE - RUN mongod service first !!

- Mac OS / linux: 
```
 $ ./installAll.sh
 $ ./runAll.sh
```
- Windows: (not tested)
```
    installAll(windows).bat
    runAll(windows).bat
```

### run manually (the order it's important)
``` 
    run mongod service  
    run python-server
    run node-server 
    run client
```
